# [WIP] TaroStats

Website health monitor with the Google Appengine + Cloud Functions


## How to develop

```
$ git clone git@github.com:miya0001/tarostats.git && cd tarostats
$ npm install
$ npm run setup
```

Unit testing

```
$ npm test
```

## Deploy

```
$ npm run deploy
```

Run following to see logs.

```
$ npm run log
```
